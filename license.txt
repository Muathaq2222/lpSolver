The "lpSolver" is a GUI developed using TKINTER package. The GUI takes input from user (with the help of SYMPY) that is required to solve an LP. Internally, PULP package is called to develop an LP. Then the LP is solved via GLPK solver. All repositories used in this application are open source. You are allowed to modify and reuse the GUI. Kindly do not remove this note from the software.
